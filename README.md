# Коммуникационное приложение

## Описание
Полнофункциональное приложение для общения с поддержкой:
- Подключение по уникальному 10-значному коду
- Текстовый чат
- Аудиозвонки
- Видеозвонки
- Обмен файлами (фото, видео, аудио, документы)
- Автоматическое сохранение файлов в папку загрузок

## Установка

### Требования
- Python 3.8+
- PyQt5
- OpenCV
- PyAudio
- Другие зависимости из requirements.txt

### Установка зависимостей
```bash
pip install -r requirements.txt
